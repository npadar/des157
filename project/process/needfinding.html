Observe the World Wildlife Fund Website ( worldwildlife.org )

  1. Looked for hover effects - expected hover but not present in some situations
  2. Interacted with popup that appeared on screen after 10 seconds
  3. Interacted with sliders
  4. Read a lot about the history and news of the organization
  5. Didn't realize could click on certain elements because there was no hover effect
  6. Clicked on tabs, which automatically scrolled down to the correct section on the page
  7. Checked for responsiveness

Feedback

  1. Liked how there was a lot of information and a range of projects
  2. Well-organized layout
  3. Sometimes there was too much information - felt overwhelmed
  4. Clicked into a sub-site ( support.worldwildlife.org ) which confused the user on why there was a login page

Brainstorm

  1. Needs to have hover effects everywhere so the user knows which items are clickable
  2. Needs to be able to organize information in a way so it does not overwhelm the user
  3. Needs to be able to provide accurate and up-to-date information
  4. Needs to provide history and validity of organization
  5. Needs to have clean layout (grid)
  6. Needs to have clear navigation so the user doesn't get lost in the vast information
  7. Needs to be responsive
  8. Needs to have donate button
  9. Needs to have strong splash images
  10. Needs to be able to affect the user to take action
